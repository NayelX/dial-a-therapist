# Visitor User Manual

## 1. Purpose
This guide is for regular site visitors, clients, guardians, and anyone interacting with public pages.

## 2. What You Can Do
- Browse service information
- Learn about the team and mission
- Submit appointment requests
- Contact the team
- Read Community Impact stories

## 3. Navigating the Site
Main pages typically include:
- Home
- About
- Services
- Community Impact
- Contact
- Appointment Request

If a page is missing, use the navigation menu or return to Home.

## 4. Booking an Appointment
1. Open Appointment Request page.
2. Fill in personal, emergency contact, and appointment details.
3. Provide accurate phone/email so the team can reach you.
4. Submit the form.
5. Wait for admin confirmation.

## 5. Understanding Appointment Status
- Pending: request received, under review.
- Confirmed: accepted and scheduled.
- Cancelled: request declined or withdrawn.

You may be contacted directly by phone or email for updates.

## 6. Using the Contact Page
1. Enter your name, email, subject, and message.
2. Submit the form.
3. A team member responds through provided contact details.

## 7. Reading Community Impact Stories
- Stories are shown as cards.
- Each card has a main image.
- If a story has multiple images, thumbnail previews appear below the main image.
- Click a thumbnail to preview that image.
- Use Read More / external link to view more details.

## 8. Mobile Use Tips
- Use portrait orientation for easier form completion.
- Ensure stable connection before submitting long forms.
- Double-check details before tapping submit.

## 9. Troubleshooting
### Form Not Submitting
- Check required fields.
- Confirm internet connection.
- Refresh and retry.

### No Confirmation Yet
- Allow response time.
- Check spam/junk folder for email replies.
- Contact the team via Contact page or official phone.

### Images Not Loading
- Refresh page.
- Try another network.
- Retry after a few minutes.

## 10. Privacy Note
Only provide information relevant to your care request. Avoid sharing sensitive details outside official forms/channels.
